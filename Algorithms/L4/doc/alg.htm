Управление алгоритмом


Кнопка Пуск запускает алгоритм. Во время работы алгоритма с графом будут происходить
изменения, которые будет видно на виджете отображения графа и в таблице результатов.
Закончится работа сообщением "Конец".

Кнопка Сброс сбрасывает начальные параметры и результаты работы алгоритма

Ползунок Скорость при смещении вправо увеличивает время между итерациями алгоритма

Флажок Пошагово запускает только один шаг алгоритма по нажатию кнопки "Пуск"

Алгоритм Форда-Фалкерсона ищет максимальный поток через сеть. Веса на графе отображаются
так: "Оставшаяся пропускная способность ребра" / "Вес ребра". При построении потоков часть
перед знаком "/" будет уменьшаться.
Потоки красятся в цвета от красного до фиолетового, согласно распределению оттенков в HSV.
Каждый следующий поток будет дальше по оттенку от красного. При построении потока
запускается анимированный шарик такого же цвета, следующий по пути потока.
